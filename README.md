# Excel

# shortkey 
Ctrl+C/V/X (Copy/Paste/Cut),
Ctrl+Z/Y (Undo/Redo), Ctrl+S (Save),
F2 (Edit Cell), F7 (Spell Check), and Ctrl+D/R (Fill Down/Right) for quick actions, while function keys (F1-F12) and combinations with Alt/Shift offer deeper menu access and specific functions like F12 (Save As) or Alt+F1 (Insert Chart).
